HAUPTAENDERUNG: Ein Button, eine Funktion
  → Der gruene Button "Gespeicherte Daten laden" wurde ENTFERNT
  → Der blaue Button "Kundendaten laden" ist jetzt der EINZIGE Lade-Button
  → Er macht ALLES in einem Schritt:
    1. Ordnerstruktur vom Kunden komplett laden
    2. "Kunden-Daten" Unterordner finden
    3. Excel-Listen herunterladen
    4. Parser startet automatisch (3 Listen: Stammdaten, Positionen, Raeume)
    5. Daten werden fuer alle Module bereitgestellt
    6. Navigation zur Daten-Uebersicht (zum Pruefen)
    7. Von dort weiter zur Modulwahl
