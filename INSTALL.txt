ÜBERBLICK
─────────
Dieses ZIP enthält das neue Homescreen-Icon für die TW Business
Suite. Das bisherige Icon (abgeschnittener Logo-Ausschnitt) wird
ersetzt durch ein klares Design: weiße Fliese auf rotem Grund
mit schwarzem "TW" und dem charakteristischen roten Punkt.

Das Icon ist direkt in die index.html eingebettet (Base64) —
keine zusätzlichen Dateien auf dem Server nötig.


ENTHALTENE DATEIEN
──────────────────
  index.html              Frisch gebaute App mit neuem Icon
                          → Diese Datei ersetzt die alte
                            index.html auf GitHub Pages

  index-template.html     Aktualisiertes Template
                          → Basis für zukünftige Builds
                          → Ersetzt die alte Version im Repo

  TW-Icon.svg             Quelldatei des Icons (editierbar)
                          → Nur für Archivzwecke, muss NICHT
                            auf GitHub hochgeladen werden


INSTALLATION AUF GITHUB PAGES
─────────────────────────────
1. Auf GitHub einloggen und ins Repo der TW Business Suite
2. Die alte "index.html" löschen oder überschreiben mit der
   neuen index.html aus diesem ZIP
3. Die alte "index-template.html" ebenfalls überschreiben
4. Commit mit Nachricht: "Homescreen-Icon aktualisiert
   (Variante E: Weiße Fliese)"
5. Warten bis GitHub Pages die Änderung ausgerollt hat
   (dauert meist 1-2 Minuten)


AUF DEM HANDY AKTUALISIEREN
───────────────────────────
WICHTIG: Das Handy cached das alte Icon! Um das neue Icon
sichtbar zu machen:

  1. Alte "Aufmaß-App" vom Homescreen entfernen (lange
     drücken → Entfernen)
  2. App im Browser (Chrome/Safari) öffnen
  3. Browser-Cache leeren (Einstellungen → Datenschutz →
     Browserdaten löschen)
  4. App neu auf den Homescreen legen:
     - Chrome: Menü (⋮) → "Zum Startbildschirm hinzufügen"
     - Safari: Teilen-Button → "Zum Home-Bildschirm"

Das neue Icon sollte jetzt erscheinen.


BEI PROBLEMEN
─────────────
Falls das alte Icon hartnäckig bleibt:
- Handy neu starten
- Oder: Im Browser die Website einmal im Inkognito-Modus öffnen
